Introduction à Git

Git est un système de gestion de versions distribué utilisé pour suivre les modifications dans le code source et collaborer efficacement. Il permet de revenir à des versions antérieures, de travailler en parallèle sur des branches, de fusionner des changements et de partager le travail entre plusieurs développeurs tout en conservant un historique clair et fiable.

Installation

  1. Installer Git via le gestionnaire de paquets de votre système (par ex. sudo apt install git sur Debian/Ubuntu, brew install git sur macOS).
  2. Télécharger Git depuis le site officiel : https://git-scm.com/downloads
  3. Configurer votre identité : git config --global user.name "Votre Nom" et git config --global user.email "vous@example.com".
  4. Vérifier l'installation en exécutant git --version pour confirmer que Git est correctement installé.
