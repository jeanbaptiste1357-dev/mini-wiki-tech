Commande Description
git status Affiche l'état des fichiers dans le répertoire de travail et l'index.
git add "fichier" Ajoute un fichier spécifique à l'index pour le prochain commit.
git commit -m "message" Enregistre les modifications dans l'historique avec un message descriptif.
git push Envoie les commits locaux vers le dépôt distant.
git pull Récupère les modifications du dépôt distant et les fusionne avec la branche locale.
git checkout -b "branche" Crée une nouvelle branche et bascule dessus.
